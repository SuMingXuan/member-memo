### 运行

#### 初始化 rails 项目

```bash
$ bundle install
$ yarn install
$ rails db:create
$ rails db:migrate
```

#### 运行 rails 项目

```bash
$ rails s
```

#### 开发环境运行 tailwindcss

```bash
$ rails tailwindcss:watch
```

#### 开发环境运行 shakapacker

```bash
$ ./bin/shakapacker-dev-server
```
